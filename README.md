- one team has 8

## setup client

```
cd client
npm install
```

## run client

```
cd client
vite dev
```

## use client

http://localhost:5173/

## setup server

```
cd server
bundle install
```

## run server

```
cd server
bundle exec rails s
```

## use server

http://localhost:3000/
